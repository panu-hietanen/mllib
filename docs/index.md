# mllib

A PyTorch-style autograd library written in C11. No dependencies beyond the C standard library.

Built as a learning project to understand both low-level memory management and the internals of ML frameworks. The long-term goal is a chess evaluation network, where you will be able to evaluate positions and potentially have a bot to play against in the [chess game](https://github.com/panu-hietanen/chess_engine) I wrote (also in C, using Raylib). Check out the devlog below for more details on the creation process, or the source code to build the project yourself!
